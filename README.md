
# Nino Joy | Portfolio

A portfolio website made with Next.js and TailwindCSS 


## Run Locally

Clone the project

```bash
  git clone https://github.com/nino-joy/nino-joy.github.io.git
```

Go to the project directory

```bash
  cd nino-joy.github.io
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn dev
```
